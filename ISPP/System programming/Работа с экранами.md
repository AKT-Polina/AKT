# LCD 16x2
Символьный дисплей **LCD 16x2** — это классика для проектов на Arduino. Название «16x2» означает, что у него **16 столбцов** и **2 строки**, то есть одновременно он может показать 32 символа.

## Программирование

Для работы с ЖК-экраном необходимо подключить библиотеку:
```c++
#include <Adafruit_LiquidCrystal.h>

#include <LiquidCrystal.h>

```
Управление 
● Инициализировать объект LiquidCrystal: Adafruit_LiquidCrystal lcd_1(0); 
● Установить размер дисплея: lcd.begin(16, 2);

Методы для работы: 
сlear() - очищает экран 
setCursor(символ, строка) - установка курсора print() - вывод значения на экран setBacklight(1) — отключает подсветку SetBacklight(0) — включает подсветку
scrollDisplayLeft();  — Сдвиг всего содержимого экрана влево

## Вывод картинок
Создаем массив байт с картинкой и выводим на экран
нарисовать картинку можно с помощью https://pkolt.github.io/bitmap_editor/
Пример: 
```c++
```
#include <LiquidCrystal.h>

LiquidCrystal lcd(12, 11, 5, 4, 3, 2);

byte pic[8] = { 0b00000, 0b01010, 0b11111, 0b11111, 0b11111, 0b01110, 0b00100, 0b00000 };

void setup() {
  lcd.createChar(0, pic); // Записываем картинкку в память под индексом 0
  lcd.begin(16, 2);
  lcd.setCursor(16, 0);     // Ставим курсор за правую границу экрана
  lcd.write(byte(0));
  lcd.print("Hello World");
  lcd.write(byte(0));
}

void loop() {
 
}

# Подключение кнопки

Чтобы подключить кнопку без внешнего резистора, нужно ==использовать **встроенный подтягивающий резистор** самой Arduino==. Это делается одной строчкой в коде.

1. Схема подключения (самая простая)

2. Один контакт кнопки соедините с **GND** (земля).
3. Второй контакт кнопки соедините с любым цифровым пином (например, **пин 2**).  

В блоке `setup()` при настройке пина нужно указать специальный режим
```c++
void setup() {
  // Включаем внутренний резистор на 20кОм, который "притянет" пин к 5V
  pinMode(2, INPUT_PULLUP); 
}

```
Когда вы используете `INPUT_PULLUP`, логика работы кнопки становится «обратной»:

- **Кнопка НЕ нажата** — на пине будет **HIGH** (1), так как внутренний резистор подтягивает его к питанию.
- **Кнопка НАЖАТА** — на пине будет **LOW** (0), так как ток уходит в землю.
```c++
if (digitalRead(2) == LOW) {
  // Кнопка нажата! Делаем что-то...
}
```
