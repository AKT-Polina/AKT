
**Задание №1: «Детектор замирания UI» (WinForms/WPF)**

- **Задача:** Создать окно с двумя таймерами: `System.Windows.Forms.Timer` и `System.Threading.Timer`. Оба должны обновлять свои текстовые метки (Label) каждую секунду, показывая текущее время.
- **Цель:** Добавить кнопку «Тяжелый расчет», которая выполняет `Thread.Sleep(5000)`.
- **Контрольный вопрос:** Почему один таймер замер во время расчета, а второй (после правильной настройки через `Invoke`) продолжил идти?

**Задание №2: «Сравнение точности» (Консоль)**
[https://habr.com/ru/companies/tbank/articles/454058/]()
- **Задача:** Написать программу, которая замеряет реальный интервал срабатывания таймера. Задать интервал 10 мс. Прогнать 100 итераций и вычислить среднее отклонение.
- **Сравнить:** Использовать `DateTime.Now` для замера и `Stopwatch`.
- **Контрольный вопрос:** Почему средний интервал оказывается ~15.6 мс, хотя заказывали 10 мс?

**Задание №3: «Безопасный асинхронный цикл» (Консоль)**

- **Задача:** Реализовать фоновую задачу с помощью `PeriodicTimer`, которая раз в 2 секунды имитирует запрос к базе данных (`await Task.Delay(3000)`).
- **Цель:** Доказать, что итерации не «наслаиваются» друг на друга, несмотря на то, что работа длится дольше интервала.

**Задание №4: «Аналитик логов без памяти»**

- **Задача:** Создать текстовый файл размером 500 МБ (заполнить случайными строками). Написать две программы для поиска в нем ключевого слова (например, "ERROR"):
    1. Используя `File.ReadAllLines` (загрузка всего в память).
    2. Используя `StreamReader` (построчное чтение).
- **Цель:** Открыть диспетчер задач и сравнить потребление оперативной памяти (RAM) во время работы обеих программ.

**Задание №5: «Шифровальщик-трубопровод»**

- **Задача:** Реализовать метод, который принимает абстрактный `Stream`, читает из него данные, делает простой XOR-сдвиг каждого байта и записывает результат в другой `Stream`.
- **Цель:** Продемонстрировать универсальность: запустить этот метод сначала для пары `FileStream` -> `FileStream`, а затем для `MemoryStream` -> `Console.OpenStandardOutput()`.

**Задание №6: «Скользящее среднее (Windowing)»**

- **Задача:** Написать программу, которая имитирует поток данных от датчика температуры (случайные числа раз в 100 мс).
- **Цель:** Использовать `IAsyncEnumerable<int>` для генерации данных и в цикле `await foreach` выводить среднее значение только за последние 10 полученных чисел (окно из 10 элементов).
